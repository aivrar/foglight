Foglight · Settings

Panels

Toggle the panels you want visible. The map, tickers, earthquakes, and severe weather are always on — they are the focus of foglight. Everything below can be hidden.

Watchlist — keyword alerts

One keyword or phrase per line. Any conflict-wire, humanitarian-sitrep, or news-ticker item matching any keyword is highlighted; new matches trigger the breaking-event banner. Case-insensitive substring match.

Map annotations

Right-click any point on the world map to pin a labeled marker (your own intel notes). Pins persist across launches and survive reboots. Click an existing pin to remove it.

Live TV

Foglight embeds free YouTube live streams of major news networks. Audio is muted by default (browser autoplay rule). Click the unmute button inside the YouTube player to hear it. Picking a station here also makes it the default on next launch.

Out-of-the-box data

Foglight ships these sources live with no signup: GDELT (world events & conflict), USGS (earthquakes, volcanoes), NWS (US severe weather), NOAA NHC (tropical cyclones), NOAA SWPC (space weather), ReliefWeb (humanitarian sitreps), mempool.space, Wikipedia EventStreams, GitHub events, ISS, CoinPaprika, Frankfurter (forex), SEC EDGAR, Hacker News, Reddit.

Optional API keys

Each key unlocks a panel. Signup is free at every link. Keys live only on disk inside this foglight instance — nothing leaves your machine.

Live global ship positions on the world map. Free key at aisstream.io.

OAuth2 client credentials. Create an account at opensky-network.org, then create an API client in your account.

Extends Severe Weather beyond the US plus global radar tiles. Free key at openweathermap.org/api.

not set

FRED · Macro indicators

Fed funds rate, 10Y yield, CPI, unemployment, VIX. Free key at fred.stlouisfed.org.

Finnhub · Stock markets

Indices, watchlist, earnings, market news. Free key at finnhub.io.

not set

Ambient audio

Subtle audio cues fire when significant events happen. Off by default. All sounds are synthesized on the fly — no audio files included.

Master · ambient audio

Top-level toggle. Everything below is muted while this is off.

Earthquakes ≥ M4.0

Pitch scaled to magnitude.

Soft chime when NWS issues a tornado warning.

Hurricane advisories

Subdued horn.

Bitcoin block mined

Quiet thunk on each new block.

About

Foglight runs entirely on your machine as a local Windows desktop app. The bundled server listens only on 127.0.0.1, and all caches, settings, and saved keys live under %LOCALAPPDATA%\Foglight\.

Close foglight

Same as the X button — stops the local server and closes the app.
